跟著可愛小動物一起學習 Git 的核心指令吧！

`git add` 🧺

將檔案從工作目錄 (Working Directory) 加入暫存區 (Staging Area)。想像成把要帶出門的東西放進購物籃裡。

`git commit` 📝

將暫存區的變更提交 (Commit) 到本地儲存庫 (Local Repository)。就像是為購物籃裡的物品蓋上一個獨一無二的時間戳記和備註。

`git push` 🚀

將本地儲存庫的變更推送 (Push) 到遠端儲存庫 (Remote Repository) (例如 GitHub)。就像發射火箭將您的作品分享給大家。

`git pull` 🎣

從遠端儲存庫抓取 (Fetch) 最新變更並合併 (Merge) 到您的本地分支。就像拉起魚竿，將最新的漁獲帶回家。

`git revert` ⏪

建立一個新的 Commit 來撤銷 (Revert) 某個舊 Commit 的變更。這是一個安全的「時光倒流」方式，不會抹除歷史紀錄。

`git checkout` 🚪

切換 (Checkout) 到不同的分支 (Branch) 或特定的版本。就像打開不同的門，進入不同的專案狀態或平行時空。

`git merge` 🤝

將一個分支的變更合併 (Merge) 到另一個分支。就像兩條小河流匯合成一條大河，共同向前流動。

學習 Git 很有趣！祝您玩得開心！ 🎉
